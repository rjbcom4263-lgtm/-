# 사주팔자 만세력 프로젝트 노트

React 19 + TypeScript + Vite 기반 사주 만세력 웹앱 (`saju-service-mvp`).
계산 라이브러리는 `manseryeok`(기준)과 `ssaju`(신강약·격국·용신 비교용) 두 개를 함께 사용한다.

## 주요 기능

- **명식 계산** — 생년월일시(양력/음력, 윤달) 입력 → 사주 네 기둥, 오행 분포, 지장간, 12운성, 12신살 표시
- **신살·길성 판정** — 천을귀인, 문창귀인, 괴강살, 백호대살, 도화살, 공망 등 19종. 핵심(core)/참고(reference) 신뢰도 구분
- **합충형파해 관계** — 천간합·충, 지지 육합·삼합·방합·충·형·파·해·원진 감지
- **운세** — 대운(방향·시작나이), 세운 10년, 월운 12개월 + 오늘/주간/월간 운세 점수(십성·오행·합충 기반)
- **만세력 달력** — 월 단위 42칸 그리드, 일진 간지·음력 날짜·절기 표시
- **리포트** — `interpretationPacks.ts`(약 3,500줄 해석문)로 초보자용 문장 리포트 생성 (동적 import로 지연 로딩)
- **프로필** — 여러 명 저장(localStorage), URL 쿼리로 공유 링크 생성

### sajuEngine 로직 흐름

1. `calculateFourPillars`(manseryeok)로 네 기둥 계산 + `calculateSaju`(ssaju)로 신강약/격국/용신
2. 기둥마다: 십성(천간·지지), 지장간, 12운성(일간×지지 조견표), 12신살(연지 삼합 그룹 기준), 신살 목록
3. 관계 탐지: 쌍 관계(합·충·파·해·원진)는 두 글자가 모두 존재하면 성립, 삼합·방합은 세 글자 전부 필요, 형은 인사신/축술미 전체 + 자묘 + 자형(같은 글자 2개)
4. 오행 분포: 8글자 중 개수 → 0%=부족, 37.5%↑=과다
5. 신살 판정 근거(수정 후):
   - 역마살·화개살·도화살·고신살·과숙살 → 연지 또는 일지 삼합 그룹 기준
   - 천문성 → 묘미술해가 일지·시지에 있을 때만
   - 정록·문창귀인·천을귀인·금여성·태극귀인·홍염살·양인살 → 일간 기준 조견표
   - 괴강살·백호대살·현침살 → 특정 간지 조합
   - 귀문관살·천라지망 → 지지 쌍 존재 여부, 공망 → 순중공망

### dailyFortune 점수 로직

기본 62점에서 시작해 십성 부스트/드래그(±5~12), 오행 일치(+6), 원국 오행 부족/과다(±4~5), 일진과 원국 지지의 합(+4)/충·해(−6)를 더해 35~95로 클램프. 주간·월간은 일별 점수 평균.

## 검증 상태

- 테스트: vitest 9개 전부 통과 (2001-08-03 13:30 남자 = 신사·을미·무술·기미 골든 테스트 포함)
- 타입체크: strict 모드 통과
- 빌드: `tsc -b && vite build` 정상 (~240KB JS)
