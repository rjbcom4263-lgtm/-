# 사주팔자 서비스 준비 체크리스트

작성일: 2026-07-03

## 현재 완료된 기반

- 만세력 계산: `manseryeok` 기준, `ssaju` 보조 비교 구조
- 입력 검증: 없는 날짜, 음력/윤달, 자시 기준 옵션
- 사주 보기: 핵심 요약, 문서형 풀이, 정밀 리포트, 접기형 상세 근거
- 문장 개인화: 십성 10개 x 신강/중화/신약 x 4분야 조건별 조언팩
- 문장 품질 검증: 50개 고정 샘플 보고서
- 사용자 기능: 프로필 저장, 공유 링크, PDF 저장용 인쇄 버튼
- 신뢰 고지: 참고용 콘텐츠, 단정 금지, 개인정보 저장/공유 안내

## 출시 전 필수 점검

1. 계산 신뢰도
   - 절기 전후 생일 20개 이상 추가 검증
   - 23시대 출생자의 자정/자시/야자시 기준 차이 검증
   - 외부 만세력 2곳 이상과 연월일시주 비교표 작성
   - 대운 시작 나이 비교 기준 문서화
- `npm.cmd run audit:engine`으로 보조 엔진 자동 비교 보고서 재생성
   - 자동 비교 산출물: `ENGINE_COMPARISON_AUTOMATED_REPORT.md`

2. 문장 품질
   - `SAJU_DOCUMENT_SAMPLE_REPORT.md`를 매번 재생성해 빈 문장, 깨진 값, 긴 문장 확인
   - 같은 구조에서 반복되는 결론 문장을 주제별 결론팩으로 세분화
   - 건강, 투자, 결혼 관련 단정 표현 재검토
   - 초보자 풀이와 전문가 풀이의 말투 차이를 더 분명히 분리

3. UX
   - 모바일에서 첫 화면 순서 확인
   - PDF 인쇄 화면에서 입력 폼과 버튼이 빠지는지 확인
   - 용어 설명 버튼의 위치와 내용 확인
   - 핵심 요약과 문서형 풀이가 너무 길게 보이지 않는지 확인

4. 개인정보/법적 고지
   - 로컬 저장 방식과 공유 링크 포함 정보 안내 확인
   - 개인정보처리방침 페이지 추가
   - 이용약관과 책임 제한 문구 추가
   - 건강/투자/법률 판단은 전문가 상담 우선이라는 문구 유지

## 다음 개발 후보

1. 외부 만세력 비교 보고서 자동화
2. 정밀 리포트 PDF 전용 레이아웃
3. 주제별 결론팩 추가: 직업/재물/관계/생활 각각 별도 결론
4. 결과 링크 공유 시 이름 노출 여부 선택
5. 결제형 정밀 풀이 분리

## 결제 기능 메모

결제는 PG사 선택이 필요하므로 코드만으로 바로 확정하지 않는다.

- 국내: 토스페이먼츠, 아임포트/포트원, 나이스페이 등 검토
- 해외/카드 중심: Stripe 검토
- 결제 전 무료 결과와 유료 결과의 차이를 먼저 정의
- 유료 영역 후보: 정밀 상담 리포트, PDF 다운로드, 대운 상세, 연애/직업 심화 풀이
